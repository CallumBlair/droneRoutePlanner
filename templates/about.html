Drone Route Planner about page.

To use this tool please follow the instruction below:

1. Navigate to the map page, select either "Farm1", or "Farm2" in the box top right of the screen

2. Place a marker

3. Select the set start node button in the sidebar

4. Place another marker

5. Select the set target node button in the sidebar

6. Confirm the path looks correct using the purple preview line that will be on your screen

7. If your happy with the route select Request Route

8. After a few seconds a route should appear

Please be patient it may take up to a minute
